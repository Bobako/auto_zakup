Формат заказа для отправки поставщику

Заказ будет отформатирован в соотвествии с этой формой.
Служебные поля, обернутые в двойные фигурные скобки будут заменены на соответствующие значения из БД.
Фигурные скобки с символом процента - цикл, выполняющий перебор всех товаров в заказе.
Описание некоторых служебных полей:
order.id - ID заказа
order.facility.name - название заведения для которого предназначен заказ
order.facility.address - адрес заведения
product.product.name - имя товара
product.amount - количество товара
product.product.unit.designation - условное обозначения единицы измерения
Менять с осторожностью.
