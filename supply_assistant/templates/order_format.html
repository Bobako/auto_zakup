Формат замовлення для відправки постачальнику

Замовлення буде відформатовано по цій формі
Службові поля, обгорнуті в подвійні фігурні скобки будуть замінені на значення з БД
Фігурні скобки з символом відсотку - цикл, який перебирає всі товари в замовленні
Опис службових полів:
order.id - ID замовлення
order.facility.name - назва закладу для якого виконується замовлення
order.facility.address - адреса закладу
product.product.name - назва товару
product.amount - к-сть товару
product.unit.designation - умовне позначення од. виміру
